How I Made This Page

  1. Html Structure
    • Used <title>, <div>, <h1>, <ul>, <ol>, <li>, <button>, <br>, <hr>, <img>, <p>, <u>, <b> tags for layout and content.
    • The page features a sidebar for navigation and links to each section.
    • The main content area contains the title, instructions, game features, and screenshots.
    • Sections are clearly identified with ids (section1, section2, section3) so the links scroll smoothly to each part.
  2. CSS Styling
    • A retro pixel font gives the page a gaming/arcade vibe.
    • The sidebar is fixed, styled with shadows and rounded corners, and responsive to toggle buttons and keyboard shortcuts.
    • The main container has a patterned background, padding, and shadow for a “floating card” effect.
    • A GIF in the corner is fixed and partially overflowing horizontally for a fun, dynamic look.
  3. Interactive Behavior (JavaScript)
    • The sidebar toggle button and "S" key show/hide the sidebar.
    • The Back to Top button scrolls smoothly to the top.
  4. Design Choices
    • Space-themed background image and subtle SVG pattern for atmosphere.
    • Large title box and clear headings improve readability.
